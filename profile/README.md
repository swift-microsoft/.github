# Swift Microsoft

Swift implementation of the Microsoft Windows platform interface — a per-authority organization of [swift-standards](https://github.com/swift-standards), Layer 2 of the [Swift Institute](https://github.com/swift-institute) ecosystem.

## What this is

This organization hosts a single package, following the same discipline as every specification package in the ecosystem: the source document implemented as literally as possible, with parsing, validation, and formatting enforced by Swift's type system. Where several specifications govern one subject, the unifying `swift-*-standard` package lives in [swift-standards](https://github.com/swift-standards).
